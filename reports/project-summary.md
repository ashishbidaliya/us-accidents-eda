# US Accidents EDA Project Summary

## Overview

This project performs Exploratory Data Analysis (EDA) on the US Accidents Dataset (2016–2023).

The objective of this project was to analyze accident trends, identify accident-prone locations, study time-based accident patterns, and visualize geographical accident distributions across the United States.

---

# Dataset Information

- Dataset: US Accidents Dataset
- Source: Kaggle
- Time Period: 2016–2023

The dataset contains millions of accident records collected from multiple traffic and accident reporting sources.

- Compared accident reporting sources

## Geospatial Analysis

---

# Key Insights

- Accident counts decrease exponentially across cities
- Peak accident hours are:
  - 6 AM to 10 AM
  - 3 PM to 6 PM
- Weekdays have higher accident frequencies compared to weekends
- Large accident clusters exist around highly populated regions
- Some inconsistencies exist across data sources
- New York data is largely unavailable in this dataset version
